## blackjack-el - Blackjack written in Elisp

Play Blackjack in Emacs.

### Install

Load the `blackjack.el` file in your .emacs or init.el:

    (load "~/path/to/blackjack.el")

### Run it:

    M-x blackjack

If you run out of money, delete blackjack.txt.
